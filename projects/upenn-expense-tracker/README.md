# UPenn Expense Tracker

Command-line Python project for importing, updating, sorting, and exporting expenses.

## Files
- `Expense.py`
- `ExpensesLoader.py`
- `ExpensesManager.py`
- `expenses.py`
- `expenses_test.py`

## Run

```bash
python3 expenses.py
```

## Test

```bash
python3 -m unittest expenses_test.py
```
