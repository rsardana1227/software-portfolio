# Movie Trivia Java App

Java project for loading movie, actor, and rating data and running different queries against it.

## Files
- `src/MovieTrivia.java`
- `src/file/MovieDB.java`
- `src/movies/Actor.java`
- `src/movies/Movie.java`

## Compile

```bash
javac src/file/*.java src/movies/*.java src/*.java
```

## Run

```bash
java -cp src MovieTrivia
```

## Notes

JUnit test files are included in the project. Running them requires JUnit 5 to be available in your Java environment.
