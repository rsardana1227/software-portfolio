# Software Projects

This repo includes some of my favorite Python and Java projects! They cover file parsing, object-oriented design, testing, basic API usage, etc.

## Projects

### Expense Tracker
Python command-line project for loading expense data from text files, updating values, sorting results, and exporting selected entries.

### Java RPG Game
Small turn-based Java game with player classes, combat logic, and round-by-round game state updates.

### Movie Trivia Java App
Java project that loads movie and actor data and supports different lookup and query methods across the dataset.

### Resume Site Generator
Python script that reads a text resume and generates a simple HTML version.

### Cerevox Parser Client
Small Python client for sending a file to the Cerevox parsing API using environment variables for configuration.

## Notes

Each project folder has its own README with basic setup or run instructions.
